# Laundry Shop Website

Ye ek simple Laundry Shop website hai jo **HTML aur CSS** se banayi gayi hai.  
Website me laundry services, pricing aur contact details dikhaye gaye hain.

## Features
- Clean and modern layout
- Background images
- Services section with images
- Contact information

## Technologies Used
- HTML5
- CSS3

## How to View
1. Repository clone karo ya download karo  
2. `index.html` file open karo browser me
